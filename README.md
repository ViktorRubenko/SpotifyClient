# SpotifyClient

## About
Have tried to get something similar to the official Spotify App.

## Frameworks
- Alamofire
- SDWebImage
- SnapKit
- SwiftLint
- LNPopupController

## Setup

```
$ cd .../SpotifyClient/
$ pod install
```
